]]> By: SteveChervitzTrutane (@trutane) https://quoderat.megginson.com/2011/06/21/mysql-wrapper-script-variables-from-the-command-line/#comment-4254 Sat, 23 Mar 2013 00:06:45 +0000 http://quoderat.megginson.com/?p=528#comment-4254 You CAN pass variables from the command-line [1], but you have to source your .sql script instead of feeding it via STDIN. Run the mysql command like this:

mysql -h HOST -u USER -pPASS -e “set @type = ‘foo’; source getstuff.sql;”

Don’t forget the semicolons.

The sad part of this story is that you CANNOT use variables for a table, column, or database name or part of such a name[2]. Rats! That would be quite useful, wouldn’t it?
